# Agents

Put SamStack agents here.

Impeccable currently contributes skills and command-style prompts, not standalone autonomous agents.

Suggested structure:

- one folder per agent
- prompt/instructions in `README.md` or `AGENT.md`
- optional assets or examples inside that agent folder

Example:

```text
agents/
  support-agent/
    AGENT.md
  research-agent/
    AGENT.md
```
